练习 4-2：使用 POST 方式获取网页源代码

这个地址只支持 POST 请求。请使用 requests.post(url, data={...}) 或 requests.post(url, json={...}) 访问，服务端会回显你提交的数据。

示例：

import requests

data = {'name': 'kingname', 'password': '1234567'}

# Form 形式
html_form = requests.post('http://exercise.kingname.info/exercise_requests_post', data=data).content.decode()

# JSON 形式
html_json = requests.post('http://exercise.kingname.info/exercise_requests_post', json=data).content.decode()
